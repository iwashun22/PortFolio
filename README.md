# 環境の構築方法
## Django編

### Pyflowをインストール
~/
```bash
brew install pyflow
```
### Pyflow経由でライブラリーをダウンロード
```bash
# /
pyflow package
```
#### Djangoのサーバーを起動する方法
```bash
# /api
pyflow manage.py runserver
```

#### Djangoの何かの処理をする時
```bash
pyflow なにか
```
